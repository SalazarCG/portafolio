FutbolManagerPro

FutbolManagerPro es una aplicación creada en Android Studio con la librería de Jetpack Compose y en lenguaje Kotlin. Desarrollada como proyecto para fin de grado, actualmente se encuentra en fase de desarrollo de manera profesional para su publicación y venta.

Pila de tecnología y bibliotecas de código abierto.

  • Nivel mínimo de SDK 24.
  • Uso de corrutinas y flujo para operaciones asíncronas.
  • Bibliotecas Jetpack:
  • Jetpack Compose: el moderno kit de herramientas de Android para el desarrollo de UI declarativa.
  • Ciclo de vida: administrando los estados de vida del usuario ante los cambios de ciclo.
  • ViewModel: administra los datos relacionados con la interfaz de usuario garantizando la supervivencia de los datos.
  • Navegación: navController.
  • Arquitectura MVVM (Vista- Modelo de vista - Modelo) promueve la capacidad de mantenimiento.
