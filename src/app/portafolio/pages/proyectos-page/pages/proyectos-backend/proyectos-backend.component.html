Proyectos Backend

Proyectos realizados en Java. Tecnologías utilizadas para crear aplicaciones de escritorio como JavaFX y Swing con Netbeans. Conexión con MySQL e Hibernate para el mapeo de objetos relacionales. Uso del novedoso motor de base de datos otientada a objetos Db4o. Conocimientos en PostgreSQL, exist y MongoDB.

Creación de componentes API REST con SpringBoot utilizando el entorno de desarrollo IntelliJ. Este último utilizado en entorno laboral como becario durante cuatro meses en la empresa NTTData.
